Terminal — Linux & macOS

One line

Needs Claude Code and tmux (sudo apt install tmux / brew install tmux).

curl -fsSL https://codechat.live/install.sh | bash

Then open a new terminal and run claude --chat. Plain claude stays exactly as it was.

VS Code — any OS, Windows too

One line

A chat panel in your sidebar. Same worldwide room as the terminal version.

curl -L https://github.com/SamiNasry/codechat/releases/latest/download/codechat.vsix -o codechat.vsix && code --install-extension codechat.vsix

Then click the CodeChat bubble in the activity bar.

What you should know

  • Everyone who installs it is in the same room. Pick a name and say hi.
  • It's public — don't paste secrets or private code.
  • New joiners see the last 50 messages; at most ~1000 recent ones exist anywhere.
  • Open source, one small binary, no telemetry, no accounts.
